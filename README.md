# DroppingSaplings

**DroppingSaplings** is a lightweight Hytale plugin designed to make reforestation and gathering more intuitive. By ensuring that saplings drop naturally from leaves across a vast range of tree types, this mod bridges the gap between harvesting and replanting.

# ‚ú® Features

- **Universal Support:** Applied to a wide variety of trees including Oak, Birch, Redwood, Palm, Crystal, and more.
- **Weighted Drops:** Custom drop containers with specific weights for ingredients and saplings.
- **Enhanced Physics:** Implemented gathering properties for soft interactions and physics for each leaf type.

# üì¶ Requirements

- Hytale (compatible version required)
- Access to your local Hytale Mods directory
- No additional dependencies

# Installation

To manually install the **DroppingSaplings** plugin, download the `DroppingSaplings-vX.X.X.zip` file from the releases page and place it into your Hytale Mods folder. The default path is:
```
C:\Users\YourUser\AppData\Roaming\Hytale\UserData\Mods
```
# Usage

Once installed, DroppingSaplings works automatically. 
- Leaves will now drop saplings based on their specific JSON drop weights.
- Includes support for: Ash, Azure, Beech, Birch, Cedar, Crystal, Dry, Fir (Red/Snow/Tip), Oak, Palm (Oasis), Poisoned, Redwood, and Windwillow.

# Compatibility

- Designed to work with standard Hytale tree generation.
- Safe for use in singleplayer and multiplayer environments.
